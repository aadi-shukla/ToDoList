# HOW TO RUN APPLICATION ON LOCAL SERVER

# START METRO SERVER:

```bash
# using npm
npm start

# OR using Yarn
yarn start
```

# install dependencies:

```bash
# using npm
 npm install

# OR using Yarn
 yarn install
```

# run applicatio:

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```
